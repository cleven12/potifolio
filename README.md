Personal portfolio of Cleven G — CS student and backend developer from Tanzania. Built with Next.js and deployed on Vercel.

## Stack

| Layer | Tech |
|-------|------|
| Framework | Next.js 16 |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Font | Geist |
| Deployment | Vercel |

## Run locally
```bash
git clone https://github.com/cleven12/portfolio
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
